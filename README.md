# FitMon

> 건강한 PC 생활을 함께하는 스트레칭 몬스터  
> 방해하지 않는 알림과 게이미피케이션을 결합한 스트레칭 유도 서비스

## Overview

FitMon은 장시간 PC를 사용하는 사용자가 작업 흐름을 크게 방해받지 않으면서도 규칙적으로 스트레칭을 실천할 수 있도록 돕는 건강 습관 형성 서비스입니다.

현대인은 하루 중 많은 시간을 앉아서 보내지만, 좌식생활의 위험성은 일상 속에서 즉각적으로 체감되지 않습니다. FitMon은 사용자의 집중을 깨는 강한 알림 대신, 메뉴바 아이콘의 상태 변화와 캐릭터 기반 피드백을 통해 자연스럽게 스트레칭 행동을 유도합니다.

## Problem

좌식생활은 사망률, 심혈관질환, 암, 제2형 당뇨병 등 다양한 건강 위험과 연결됩니다. 그러나 사용자는 장기적 건강 위험을 즉각적으로 체감하기 어렵기 때문에 행동 변화로 이어지기 어렵습니다.

기존 건강 앱에도 한계가 있습니다.

- 건강·피트니스 앱은 초기 이탈률이 높음
- 푸시 알림과 팝업은 작업 흐름을 방해함
- 알림 피로가 쌓이면 사용자는 알림을 끄거나 앱을 삭제함
- 단순 기록 중심 기능은 지속 동기를 만들기 어려움

## Solution

FitMon은 `방해하지 않는 알림`과 `게이미피케이션`을 결합합니다.

소리나 팝업으로 사용자의 작업을 끊는 대신, 메뉴바 아이콘의 상태 변화를 통해 사용자가 자연스럽게 휴식 타이밍을 인지하도록 합니다. 또한 사용자가 스트레칭을 수행하면 캐릭터인 FitMon의 상태가 회복되고 성장하는 구조를 통해 지속적인 동기를 제공합니다.

## Key Features

### 1. 맞춤 스트레칭 수행

사용자는 현재 상태와 필요에 따라 스트레칭을 선택하고 수행할 수 있습니다.

- 부위별 스트레칭 추천
- 난이도 및 소요 시간 기반 선택
- 짧은 시간 안에 수행 가능한 동작 제공
- 작업 중 부담 없는 스트레칭 플로우

### 2. FitMon 상태 변화

FitMon 캐릭터는 사용자의 휴식 및 스트레칭 기록에 따라 상태가 변화합니다.

- 장시간 미휴식 시 피로 상태 표시
- 스트레칭 수행 시 회복
- 캐릭터 표정 및 상태 변화
- 사용자의 행동에 반응하는 피드백 구조

### 3. 스트레칭 기록 저장

사용자의 스트레칭 기록을 저장하여 건강 습관 형성을 돕습니다.

- 스트레칭 수행 기록
- 날짜별 기록 확인
- 연속 수행 여부 확인
- 간단한 진행 상황 제공

### 4. 방해하지 않는 알림

FitMon의 알림은 기존 팝업 알림과 다르게 사용자의 작업을 강제로 중단시키지 않습니다.

- 메뉴바 아이콘 상태 변화
- 비침습적 피드백
- 작업 흐름 유지
- 자연스러운 인지 유도

## Target Users

- 하루 대부분을 PC 앞에서 보내는 대학생
- 장시간 근무하는 직장인
- 건강 앱을 설치했지만 오래 사용하지 못한 사용자
- 팝업 알림과 푸시 알림을 부담스럽게 느끼는 사용자
- 가볍고 지속 가능한 스트레칭 습관을 만들고 싶은 사용자

## UX Strategy

FitMon은 사용자가 "운동을 해야 한다"는 부담을 느끼기보다, 캐릭터를 돌보는 가벼운 경험 속에서 자연스럽게 움직이도록 설계되었습니다.

핵심 UX 전략은 다음과 같습니다.

- 작업 흐름을 방해하지 않는 알림
- 캐릭터 상태 변화를 통한 직관적 피드백
- 타마고치 효과를 활용한 지속 동기
- 짧고 간단한 스트레칭 수행
- 기록을 통한 습관 형성

## MVP

FitMon의 MVP는 다음 세 가지 기능을 중심으로 구성됩니다.

1. 맞춤 스트레칭 수행
2. FitMon 상태 변화
3. 스트레칭 기록 저장

## User Test

사용성 테스트를 통해 다음 문제를 확인했습니다.

- 핵심 기능 진입 경로의 워딩이 직관적으로 전달되지 않음
- 캐릭터 상태 변화가 충분히 인식되지 않음
- 일부 화면에서 정보 위계가 불명확함

이를 바탕으로 기능 표현, 상태 피드백, 화면 정보 구조를 개선할 예정입니다.
